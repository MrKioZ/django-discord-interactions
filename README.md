# django-discord-interactions
A django reusable application for discord interactions


## Installation

### Settings.py
```python
DISCORD_INTERACTION_PUBLIC_KEY = ''
DISCORD_INTERACTION_APPLICATION_ID = ''
DISCORD_INTERACTION_BOT_TOKEN = ''

INSTALLED_APPS = [
    ...
    'discord_interactions',
    ...
]

```
